## TELSA.Messaging
TELSA.Messaging is a multi-provider Chat Bot library.

It can help you to create a cross-platform Chat Bot.

Such as Facebook, LINE, WeChat and etc...

## Features
* Send messages in LINE Messaging API.

## Minimum Requirements
* .NET Standard 2.0
* .NET Core 2.0
* .NET Framework 4.6.1
